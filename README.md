A resume serving, largely static website, hosted in GitHub Pages with CloudFlare workers to manage the LLM calls and responses. 

The resume content comes from content/resume.md . Before/while committing it, site/resume.html will be autogenerated using scripts/build.sh


# Process to deploy the worker

1. Create the Turnstile widget and copy the site key and site secret. The site key goes into `wrangler.toml` and `index.html`
2. The secret needs to be entered via `npx wrangler secret put TURNSTILE_SECRET_KEY`. Same for `GROQ_API_KEY`
